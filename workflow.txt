# Git Workflow for Task App Project

152531 MAXWELL KIMANI ICS 2C

1. Initialize the local repository
   git init

2. Add all project files to staging
   git add .

3. Commit the initial version
   git commit -m "First commit"

4. Publish the local repo to the remote GitHub repository

5. Push the initial commit to GitHub
   git branch -M main
   git push -u origin main

6. Added PHPMailer to the project

7. Make changes to mail file and signup form

8. Stage the changes
   git add .

9. Commit
   git commit -m "Added email validation, welcome email, and numbered user list"

10. Push the updated project to GitHub
   git push
